# Roth IRA Calculator

[rothiracalculator.app](https://rothiracalculator.app/) is a free, open-source Roth IRA calculator that helps you plan your retirement savings. It runs entirely in the browser with no backend, no accounts, and no ads.

The tool includes four calculators in one page: a growth projector that charts your Roth IRA balance year by year with compound interest, an eligibility checker based on 2025/2026 IRS income limits and filing status, a Roth vs Traditional IRA comparison using a three-step tax model (including tax savings reinvestment), and a Roth vs Taxable brokerage account comparison that accounts for annual dividend tax drag and capital gains on sale.

All IRS data (contribution limits, income phase-outs, catch-up contributions) is sourced from official IRS publications and updated for the current tax year. The calculator is a single HTML file with no dependencies — just open it in a browser.
